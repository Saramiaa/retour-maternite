# Retour Maternite

Ce site propose à des mamans qui sortent de la maternité, des gardes de Nurse diplômé.

Une maman ou un papa, doit s'inscrire pour pouvoir réserver une garde.
Aprés inscription, la personne à accés à son profil où elle peut modifier ses informations et gérer ses reservations.

Une plateforme admin a été aussi conçu pour la gestion des inscrits et des réservations effectué par les inscrits.

# Avant le lancement du site:
  - Importer la BDD (booking.sql)
  - Changer le fichier 'database.php' (application/config/database.php) avec vos identifiants si nécessaire
